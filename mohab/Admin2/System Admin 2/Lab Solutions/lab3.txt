1. Find out what device node the /boot partition is

df -l
output:
Filesystem          1K-blocks     Used Available Use% Mounted on
/dev/sda4             1998672   283848   1593584  16% /boot
then:
cat /proc/partitions
output:
major minor  #blocks  name
8        4    2097152 sda4

or 
lsblk

2. Estimate the size in bytes of a level-zero dump for /boot

df -H
output:
Filesystem           Size  Used Avail Use% Mounted on
/dev/sda4            2.0G  278M  1.6G  16% /boot

so level 0 dump will be 2 * 10e9 bytes

3. Back up the data of /boot to a dump file in /var/tmp/dumpfile

<Note that I first installed dump package>
dump -0u -f /var/tmp/dumpfile /boot

4. Look in the /etc/dumpdates file and see how the dump command recorded the timestamp of the full backup

cat /etc/dumpdates
output:
/dev/sda4 0 Tue Feb  9 16:10:07 2021 +0200
