4. Bring your interface down.

ifdown eth0 ---> (to bring it up again, use "ifup eth0" cmd)

5. Configure your network card to have static IP.

vi  /etc/sysconfig/network-scripts/ifcfg-eth0
(You only need to change IPADDR, GATEWAY and DNS1)
# static IP address on CentOS 7 or RHEL 7#
HWADDR=00:08:A2:0A:BA:B8
TYPE=Ethernet
BOOTPROTO=none
# Server IP #
IPADDR=192.168.2.203
# Subnet #
PREFIX=24
# Set default gateway IP #
GATEWAY=192.168.2.254
# Set dns servers #
DNS1=192.168.2.254
DNS2=8.8.8.8
DNS3=8.8.4.4
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
# Disable ipv6 #
IPV6INIT=no
NAME=eth0
# This is system specific and can be created using 'uuidgen eth0' command #
UUID=41171a6f-bce1-44de-8a6e-cf5e782f8bd6
DEVICE=eth0
ONBOOT=yes

then   -> systemctl restart network
verify -> ifconfig or ip a s eth0

ifup eth0
